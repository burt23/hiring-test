# Sam's Storage Space Manager
Sam sells sheets but is having trouble keeping track of the stock across his three storage containers. He'd like a way to visualize how many King, Queen, Full, and Twin sheets he has in each container AND would like a way to manage his stock in an intuitive way. 

### Objectives
* manage inventory of sheets across mulitple storage containers
* visually represent the quantity available for each size in each container
* add functionality to add and delete an item in a container
* ensure the application is responsive

### Bonus Objectives
* add an error message when stock is close to zero
* add an animation when stock reaches 0
* add color to the inventory details
